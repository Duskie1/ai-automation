# Design Specification: Retell AI (Visual Audit)

## 1. Design Philosophy & Visual Identity
Retell AI utilizes a **sophisticated, high-tech aesthetic** that blends editorial elegance with modern SaaS functionality. The design language is characterized by a "Dark Mode" hero section transitioning into a clean, light-themed body, emphasizing its "future of call centers" positioning. It uses high-contrast typography and fluid, organic gradients to convey a sense of advanced AI technology.

| Attribute | Description |
| :--- | :--- |
| **Style** | Editorial Tech / Modern SaaS |
| **Tone** | Innovative, Premium, Reliable |
| **Visual Hierarchy** | Large serif headings for brand storytelling, clean sans-serif for UI |

---

## 2. Color System
The palette is built around a deep, atmospheric blue/black for the hero and a crisp white for the functional sections.

| Token | Value (Hex/RGB) | Usage |
| :--- | :--- | :--- |
| **Hero Background** | Deep Blue/Black Gradient | Hero section, primary brand impact |
| **Primary Background** | `#FFFFFF` | Main body and functional sections |
| **Primary Text** | `#333333` / `#000000` | Headings and primary body copy |
| **Accent Color** | Royal Blue / Indigo | Links, icons, and primary CTAs |
| **Secondary Background** | `#F8F9FA` | Card backgrounds and section dividers |

---

## 3. Typography
Retell AI uses a sophisticated pairing of a serif font for brand impact and a robust sans-serif for utility.

### **Headings: Untitled Sans / Georgia (Editorial Serif)**
- **Character:** High-contrast, elegant, authoritative.
- **Usage:** Hero titles, major section headers.
- **Note:** The use of a serif font like Georgia or a custom editorial serif distinguishes it from typical "flat" SaaS designs.

### **Body & UI: System Sans-Serif**
- **Character:** Neutral, highly legible, functional.
- **Usage:** Navigation, labels, body text, documentation.

---

## 4. Layout & Grid System
- **Hero Section:** Full-bleed immersive background with centered or left-aligned high-impact typography.
- **Content Width:** Standard 1200px - 1280px max-width.
- **Sectioning:** Clear horizontal bands with alternating background colors (White vs. Light Gray) to define content blocks.
- **Bento Elements:** Uses a grid of cards for feature highlights, similar to Cal.com but with more emphasis on iconography and illustrative elements.

---

## 5. Component Patterns

### **Cards**
- **Border Radius:** `12px` to `16px`.
- **Style:** Subtle borders (`1px solid #E5E7EB`) with very soft shadows or flat backgrounds.
- **Interactive:** Hover states often include a slight lift or border color change.

### **Buttons**
- **Primary CTA:** Rounded-pill or slightly rounded (`8px`), high-contrast (Black or Deep Blue background).
- **Secondary:** Ghost buttons or text links with trailing icons (e.g., `chevron_right`).

### **Interactive Demos**
- **Style:** Embedded, simplified UI components (like the "Try Our Live Demo" widget) that mimic real application interfaces.

---

## 6. Imagery & Motion
- **Gradients:** Soft, "aurora" style gradients in the hero section.
- **Icons:** Clean, stroke-based icons, often paired with a secondary color or background shape.
- **Social Proof:** High-quality photography for testimonials, often presented in a slider or grid with prominent quotes.

---

## 7. Key Takeaways for AI Designer
1. **Editorial Contrast:** Mix a premium serif for headings with a clean sans-serif for UI.
2. **Immersive Hero:** Start with a high-impact, dark/gradient hero before moving into a clean light layout.
3. **Pill-Shaped Elements:** Use rounded-pill shapes for tags and primary buttons to soften the tech-heavy feel.
4. **Functional Minimalism:** Keep the body sections extremely clean, using whitespace to separate complex technical information.
